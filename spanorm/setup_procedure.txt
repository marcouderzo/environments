For CBP machines.

Steps:
1. Download Miniforge3 bash script for correct CPU architecture
    a. check the CPU architecture with: uname -m
    b. get the link from: https://github.com/conda-forge/miniforge/releases
    c. download with wget 
    d. install with bash Miniforge3-*.sh
2. Create the spanorm conda environment with: conda env create -f spanorm.yml
3. Activate the environment with: conda activate spanorm
4. Find the opencl header directory with: find ~ -iname "*opencl*" -print
5. Export the OPENCL_INC bash variable, it should point to the subdirectory "include" of the opencl header directory named "opencl-headers-***", e.g.:
    export OPENCL_INC="/home/muderzo/miniforge3/pkgs/opencl-headers-2025.06.13-h5ad3122_0/include"
6. Launch R
7. Install the packages SpaNorm and gpuR:
    a. devtools::install_github("bhuvad/SpaNorm@gpu")
    b. install.packages("gpuR")
